Irregular Grid Interpolator - интерполяция табличных данных



Версия 1.2.3 (май 2015г.)

Разработчик Смехов А.Н.



Данная программа предназначена для нахождения значений табличной функции двух переменных в узлах регулярной сетки на основе
известного набора точек. Исходный набор точек может быть задан как на регулярной равномерной сетке, так и на нерегулярной сетке.



Для интерполяции данных программа использует методы библиотеки ALGLIB (©Sergey Bochkanov, ALGLIB Project, www.alglib.net), 
распространяемой по лицензии GNU/GPL (файл alglibnet2.dll). Для визуализации поверхности используется библиотека OpenGL, доступ к 
которой производится через функции библиотеки OpenTK (файл OpenTK.dll).



Для работы программы требуется Microsoft .NET Framework версии 3.5 или выше.

РАБОТА С ПОВЕРХНОСТЬЮ
1) Навигация с помощью мыши.
При зажатой ЛКМ можно вращать область построения, размещая её под удобным для обзора углом. При зажатой ПКМ можно перемещать область построения
в плоскости экрана. С помощью колёсика мыши можно увеличивать и уменьшать область построения.
2) Выделение точек нажатием ЛКМ.
Выделенная точка подсвечивается белым маркером, возле которого отображаются координаты этой точки. Значения выделенных узлов интерполированной
поверхности можно изменять нажатием клавиш «+», «-» на 1%, а с зажатой клавишей Shift – на 5%. Выделенные точки из исходного набора можно удалять
нажатием клавиши Delete. При этом автоматически будет выполнена повторная интерполяция, и результаты будут сразу же отображены на графике.
Выделение с точек снимается нажатием клавиши Escape.

ФОРМАТ ХРАНЕНИЯ ДАННЫХ
1) Файл исходных данных должен содержать набор строк, значения в которых должны быть разделены одним из следующих символов:
  1. символ табуляции
  2. символ пробела
  3. символ ";"
2) Файл осей должен содержать всего две строки, в которых хранятся значения первого и второго аргумента соответственно. В качестве разделителей
должны использоваться те же символы, что и для файла исходных данных.
3) Файл конфигурации представляет собой простой текстовый файл, хранящий построчно следующие значения:
  1. полное имя файла исходных данных
  2. полное имя файла осей
  3. тип исходного набора данных
    3.1. 0 - нерегулярная сетка
      3.1.1. режим чтения таблицы исходных данных (1 - по столбцам, 2 - по строкам)
      3.1.2. количество строк/столбцов для пропуска
      3.1.3. номер столбца/строки со значениями первого аргумента
      3.1.4. номер столбца/строки со значениями второго аргумента
      3.1.5. номер столбца/строки со значениями функции
      3.1.6. метод интерполяции (0 - линейный, 1 - квадратичный)
    3.2. 1 - регулярная сетка
      3.2.1. расположение данных для аргумента X (True - в строке, False - в столбце)
      3.2.2. номер строки/столбца со значениями аргумента X
      3.2.3. начало диапазона столбцов/строк со значениями аргумента X
      3.2.4. конец диапазона столбцов/строк со значениями аргумента X
      3.2.5. расположение данных для аргумента Y (True - в строке, False - в столбце)
      3.2.6. номер строки/столбца со значениями аргумента Y
      3.2.7. начало диапазона столбцов/строк со значениями аргумента Y
      3.2.8. конец диапазона столбцов/строк со значениями аргумента Y
      3.2.9. начало диапазона строк со значениями функции Z
      3.2.10. конец диапазона строк со значениями функции Z
      3.2.11. начало диапазона столбцов со значениями функции Z
      3.2.12. конец диапазона столбцов со значениями функции Z
